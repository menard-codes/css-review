How to choose between CSS Grid and Flexbox

CSS Grid is for Layout, Flexbox is for Alignment of Elements

CSS grid came up to help solve layout issues using 2D way. Learn first flexbox before grid, because when you need alignment, it's Flexbox that should be used.

Consider Flexbox when:

  1. You have a small design to implement
  2. You need to align elements
  3. You need a content first design

Consider CSS Grid when when:

  1. You have a complex design to implement
  2. You need to have a gap between block elements
  3. You need to overlap elements
  4. You need a layout first design
