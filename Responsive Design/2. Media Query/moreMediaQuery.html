Media Query

Media Queries are commonly associated with CSS, but they can be used in HTML and JavaScript as well

HTML

Tags can use Media Queries. There are several reasons to use them, here's an example:

-In a link tag where we import a specific css file depending on the media query condition (i.e. we have a css for all instances, a css for mobile view, for desktop view, etc.) <link rel='stylesheet' href='small.css' media='(min-width: 20em)'

Another one is with picture tag where on the srcset, is specifies which version of the image to download based on sreen size

JavaScript

JS Media Query acts pretty much like in CSS. We do media queries here via window.matchMedia( '( min-width: 768px; )' )

Anatomy of a Media Query

@media - at-rule screen - media type (min-width: 320px) - media feature (many more on css tricks media query) and - logical operators -and -or (comma ',') -not

Avoid Complicating the codebase

Assuming is really dangerous, so when you design, make sure to develop and think about your products while leaving assumptions behind and use media queries to make sure that your content is displayed correctly in any contact and before any user.

Nesting and complex decision making

We can nest media queries and perform more complex and even nested logic @media (min-width: 20em), not all and (min-height: 40em) { @media not all and (pointer: none) { ... } @media screen and ( (min-width: 50em) and (orientation: landscape) ), print and ( not (color) ) { ... } }
